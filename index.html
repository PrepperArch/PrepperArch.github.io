Apocalypse OS

Repository

Apocalypse OS repository setup. Create the following file /etc/pacman.d/apocos.mirrorlist with following content:

# # Apocalypse OS Mirrorlist # Server = https://prepperarch.github.io/apocos-repo/$arch

And add the repo to /etc/pacman.conf as: [apocos] SigLevel = Optional TrustAll Include = /etc/pacman.d/apocos.mirrorlist Update the repositories using the pacman -Sy and then install the packages you are interested in.
