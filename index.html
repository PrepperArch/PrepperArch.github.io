Apocalypse OS

What is Apocalypse OS ?

Apocalypse OS is an tailored ArchLinux distribution for hacking and software defined audio (SDR). It main focus is to provide an Raspberry PI installation media to build a rugged CyberDeck computer with the tools of your needs.

Repository

Apocalypse OS repository setup. Create the following file /etc/pacman.d/apocos.mirrorlist with following content: 


#
# Apocalypse OS Mirrorlist
#
Server = https://prepperarch.github.io/apocos-repo/$arch

And add the repo to /etc/pacman.conf as: 


[apocos]
SigLevel = Optional TrustAll
Include = /etc/pacman.d/apocos.mirrorlist

Update the repositories using the pacman -Sy and then install the packages you are interested in.
